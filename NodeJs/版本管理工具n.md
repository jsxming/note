#### 安装

```javascript
sudo npm install n -g
```



#### 下载并切换

```
// 下载并切换到版本
n 16.14.1   
```



#### 查看安装的版本列表

```javascript
n ls
```



#### 其他操作

```javascript
//下载最新
n latest

//下载最新的稳定版
n lts

//删除指定版本
n rm version

//指定版本node执行脚本

n run <version> xxx
n use <version> xxx
n as <version> xxx
```
